什么是 SagaSmith？

What is SagaSmith?

一套由 AI Agent 驱动的桌面角色扮演工具集。不止是掷骰子和查规则——它是完整的 AI 地下城主。

An AI Agent-driven toolkit for tabletop RPGs. More than dice and rule lookups — it's a complete AI Dungeon Master.

快速开始

Quick Start

完整版

Full Runtime

# 安装 D&D 运行时# Install D&D runtime $ pip install "sagasmith-dnd[documents]" $ sagasmith-dnd doctor --json $ sagasmith-dnd campaign start --name "深渊之门" --edition 2024 --json # 需要语义搜索时再加 ChromaDB# Add ChromaDB for semantic search $ pip install "sagasmith-dnd[all]" $ export CHROMA_DB_PATH=~/.sagasmith/chroma_db

轻量版（Standalone）

Standalone

# 克隆仓库，切到 standalone 目录# Clone repo, switch to standalone/ $ cd standalone/ $ python portable.py doctor $ python portable.py campaign start --name "Campaign" --edition 2024 $ python portable.py roll dice --expression "2d6+3"
